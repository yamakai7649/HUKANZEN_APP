# HUKANZEN

> 🚧 このアプリは現在開発中です。  
> 時間記録・ポモドーロタイマー・ダッシュボードなどの基本機能を中心に実装しており、今後は共有ルームや公開ログ、ランキングなどの機能を追加していく予定です。

HUKANZEN は、学習・筋トレ・読書などの日々の活動時間を記録し、習慣化を支援する時間管理アプリです。  
ポモドーロタイマーや手動記録、ダッシュボードによる可視化を通じて、自分が「何にどれくらい時間を使ったか」を振り返ることができます。

現在は、個人の時間記録・習慣化機能を中心に開発しています。  
今後は、共有ルーム・公開ログ・ランキング・目標管理などを追加し、ユーザー同士が一緒に継続できる習慣化プラットフォームへ拡張していく予定です。

## 開発背景

学習や運動、読書などを継続したいと思っていても、実際にどれくらい取り組めているのかが見えないと、モチベーションを維持しづらいと感じていました。

そこで、日々の活動時間を簡単に記録し、ダッシュボードで振り返れるアプリを作ることで、継続のハードルを下げたいと考えました。

StudyPlus のように記録が積み上がる達成感を持ちながら、よりシンプルで、学習以外の活動にも使える習慣化アプリを目指しています。

---

## コンセプト

HUKANZEN は、単なる時間記録アプリではなく、ユーザーが「続けられる状態」を作ることを目的にしています。

現在は個人の記録・可視化を中心にしていますが、今後は他のユーザーと一緒に作業できる共有ルームや、公開ログ、ランキングなどを追加し、1人でも複数人でも継続しやすい体験を作っていく予定です。

---

## 現在実装している主な機能

### 時間記録

- 手動で活動時間を記録
- タイトル・カテゴリ・時間・メモを入力して保存
- 15 / 30 / 45 / 60 / 90 / 120 分のクイック入力
- 記録ごとに公開 / 非公開を選択可能

### ポモドーロタイマー

- ポモドーロ / 短い休憩 / 長い休憩の切り替え
- タイマー完了後に時間記録へ保存
- オーバータイムモード
- ゲストモードでログインなしでもタイマーを使用可能

### ダッシュボード

- 今日・今週・今月の合計時間
- 週間目標の進捗表示
- カテゴリ別の時間内訳
- 過去14週のヒートマップ
- 最近の記録一覧

### カテゴリ管理

- 学習・筋トレ・読書などのカテゴリ作成
- カテゴリごとの色設定
- デフォルトカテゴリの追加

### 記録一覧

- 今週 / 今月 / 全期間でのフィルタ
- カテゴリ別フィルタ
- 記録の削除

### プロフィール

- ユーザー名・自己紹介
- 総記録時間
- ストリーク表示

## 今後追加予定の機能

### 記録・分析機能の強化

- 記録の編集機能
- カテゴリ別の累計時間表示
- 月別・週別の推移グラフ
- 週間 / 月間レポート
- プロフィールへのヒートマップ表示
- 継続日数や達成率に応じたバッジ機能

### 目標管理

- 長期目標の作成
- 目標ごとの進捗管理
- 週ごとの振り返り機能
- 学習・筋トレ・読書など目的別の目標テンプレート

### ソーシャル機能

- 公開ログのフィード
- ユーザー同士のフォロー機能
- いいね・リアクション機能
- 週間ランキング
- ユーザープロフィールの拡張

### 共有ルーム機能

- ユーザーが集中ルームを作成・参加できる機能
- ルーム内で各ユーザーの集中時間を可視化
- 同じ目標を持つユーザー同士で一緒に作業できる仕組み
- ルームごとの累計時間・ランキング表示
- 勉強会や作業部屋として使える機能

### 継続支援

- リマインダー通知
- PWA対応
- 記録忘れを防ぐ通知
- 過去の記録をもとにした振り返り表示
- AIを使った振り返り・改善提案機能

## DB設計概要

| テーブル | 役割 |
|---|---|
| `profiles` | ユーザー名・自己紹介・ストリークなど |
| `categories` | ユーザーごとのカテゴリ管理 |
| `time_logs` | 手動記録・タイマー記録 |
| `weekly_goals` | 週間目標時間 |
| `goals` | ポモドーロ用ゴール |
| `sessions` | ポモドーロセッション記録 |

Row Level Security により、自分のデータは自分だけが操作できるように設計しています。  
また、`time_logs` の `visibility` を使って、公開ログと非公開ログを切り替えられるようにしています。
